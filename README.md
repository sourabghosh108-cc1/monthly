# Monthly Budget Planner

This project provides a **multi-platform solution** for calculating monthly budgets based on income, fixed expenses, and variable expenses.

It includes implementations for:

- Python (cross-platform script)
- Web application
- Windows executable
- Android APK

The project demonstrates **core programming concepts** such as sequencing, iteration, conditional logic, and debugging.

---

# Features

- Budget calculation based on **income and expenses**
- Handles **fixed and variable expenses**
- Demonstrates **logical error handling and debugging**
- **Multi-platform deployment**
  - Python script
  - Web app
  - Windows executable
  - Android APK

---

# Getting Started

```bash

*Clone the Repository*

git clone https://github.com/sourabghosh108-cc1/Monthly-Budget-Planner.git
cd Monthly-Budget-Planner/

*Run Python Script (Cross-Platform)*

Linux

1.Debian / Ubuntu

sudo apt update
sudo apt install python3 python3-pip -y
python3 monthly.py

2.Fedora / CentOS / RHEL

sudo dnf install python3 python3-pip -y
python3 monthly_budget.py

3.Arch / Manjaro
sudo pacman -Syu python
python monthly.py

4.macOS (Homebrew)
brew install python
python3 monthly.py

5.Android (Termux)
pkg update && pkg upgrade
pkg install python
python3 monthly.py

6.Windows

Install Python from the official website and run:

python monthly.py

⚠️ Note: Ensure Python is added to your system PATH.

```

**Debugging**

Test the program with different scenarios:

Zero income

Multiple variable expenses

Overspending situations

You can debug using:

Python’s built-in debugger (pdb)

Any IDE debugger

Helpful debugging variables:

fixed_total
variable_total
remaining_budget

Printing these intermediate values can help identify logical errors.
